# C++ CLI Task Manager

A simple C++ command-line task manager with file persistence using OOP and STL.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as completed
- Save tasks to a file
- Load saved tasks from a file

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and objects
- STL vector
- File handling (`ifstream`, `ofstream`)
- String parsing
- Modular function design
